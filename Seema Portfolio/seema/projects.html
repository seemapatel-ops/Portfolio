Project Overview

Smart Contact Manager is a comprehensive web application designed to help users efficiently manage their personal and professional contacts. Built with modern Java technologies, this application provides a secure, user-friendly platform for storing, organizing, and managing contact information with advanced features like image upload, social media integration, and search functionality.

The application allows users to create accounts, add unlimited contacts, organize them into categories, and access them from anywhere. With a responsive design and robust backend, it offers seamless contact management experience similar to a personal CRM system.

Key Features

User Authentication

Secure user registration and login with Spring Security, password encryption, and session management

Contact Management

Add, edit, delete, and view contacts with detailed information including name, email, phone, address, and more

Image Upload

Upload and manage profile pictures for contacts with automatic image resizing and storage

Search Functionality

Advanced search feature to quickly find contacts by name, email, or phone number

Contact Categories

Organize contacts into categories like Family, Friends, Work, Business, etc.

Responsive Design

Mobile-friendly interface that works seamlessly on all devices
